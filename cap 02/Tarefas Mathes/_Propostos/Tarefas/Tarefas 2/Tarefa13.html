Quem criou o JavaScript e em que ano ele foi criado?

O JavaScript foi criado por Brendan Eich, um programador da Netscape Communications Corporation. Ele desenvolveu a linguagem em 1995. Inicialmente, o JavaScript foi chamado de Mocha, depois foi renomeado para LiveScript, e finalmente recebeu o nome de JavaScript. A ideia por trás do JavaScript era permitir a criação de interatividade nas páginas da web, complementando as funcionalidades do HTML e CSS.


O que são eventos em JavaScript?Dê um exemplo simples

Eventos em JavaScript são ações que acontecem no navegador, como cliques, digitação ou movimento do mouse. Eles podem ser capturados e manipulados para executar uma função específica quando o evento ocorre.

Oque é visual Studio Code?

O Visual Studio Code (VS Code) é um editor de código gratuito e de código aberto da **Microsoft**. Ele é leve, rápido e altamente personalizável, com suporte a várias linguagens de programação. Oferece recursos como autocompletar, depuração integrada, terminal embutido e extensões para aumentar sua funcionalidade. É muito usado por desenvolvedores devido à sua flexibilidade e poder.


Oque é bootstrap?

**Bootstrap** é um framework de código aberto para desenvolvimento web que facilita a criação de sites responsivos e com design consistente. Ele oferece componentes prontos (como botões e formulários), é baseado em CSS e JavaScript, e adapta o layout a diferentes tamanhos de tela. É amplamente usado para agilizar o desenvolvimento de interfaces.
