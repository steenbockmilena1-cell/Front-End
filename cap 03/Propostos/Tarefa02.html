Estudando HTML

Nssos estudos começaram falando a respeito de um item muitosério que é a organização .Isso porque a linguaem HTML
é responável apenas pela marcação dos elementos na janelo do navegador.O HTML não cria nada,le manda buscar ou utiliza as letrinhas que você digita

Parágrafos


Para colocarmos parágrafos ,usamos os elementos "p" e "p/" .Para fazer uma quebra de linha ,mantendo o mesmo parágrafo
,usamos o elemento "br" .Temos ainda parágrafos especiais ,denominadas como títulos ,cabçalhos ou headings em que são usados elemetos
"h1" a "h6" e seus respectivos fechamentos de "/h1" a"/h6"

Imagens

Uma imagem aparece na janela do navegadorpelo o uso do elemeno "img" .esse elemento possui o atributo "src",responsável pela localização
da imagem ,você pode colocá-la dentro das tags de marcação do parágrafo
